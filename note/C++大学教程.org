** 数据的层次结构
| 层次              | 描述                                                                                     |
| 位（bit）         | 计算机中最小的数据项，可以是0或者1                                                       |
| 字符（character） | 我们称数字、字母、特殊符号为字符，计算机的字符是用于编写程序和表示数据项的所有字符的集合 |
| 字段（field）     | 字段由字符或字节组成，一个字段是可以表达意义的一段字符或字节                             |
| 记录（record ）   | 若干相关字段组成记录                                                                     |
| 文件（file）      | 文件是一组相关的记录                                                                     |
| 数据库（database) | 数据库是一个电子化的数据集合                                                                           |
